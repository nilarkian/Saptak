### The 12  "source codes" 

| Tier         | Source Code                                                                                                                                                                  |
| ------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **God Tier** | Language, Symbolism, Classification, Theory of Mind, Kinship, Reciprocity, Coalition Formation, Status Competition, Moral Sentiments, Future Simulation, Narrative, Tool Use |

Everything else on is largely downstream of those 12.

For example:

* Religion ← Narrative + Agency Detection + Coalitions
* Law ← Reciprocity + Moral Sentiments + Coalitions
* Trade ← Reciprocity + Future Simulation
* Marriage ← Kinship + Sexual Regulation
* Institutions ← Coalitions + Narrative + Status
* Myths ← Language + Narrative + Symbolism
* Prestige ← Status Competition
* Property ← Reciprocity + Future Simulation
