### Specific Patterns 

| Pattern                    | Why It Matters                |
| -------------------------- | ----------------------------- |
| Role prompting             | Controls style + domain       |
| XML / structured prompting | Stabilizes outputs            |
| Few-shot examples          | Pattern anchoring             |
| Decomposition              | Better reasoning              |
| Reflection loops           | Improves reliability          |
| Evaluation prompts         | Self-checking systems         |
| Context layering           | Better large-context handling |
